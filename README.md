# 11900 電腦軟體設計丙級 - 自動化測試系統

本專案提供針對「電腦軟體設計丙級」試題的自動化檢測工具，旨在協助應試人確認程式碼邏輯與輸出結果是否符合評分標準。支援 **C#** 與 **VB.NET** 雙語言版本。

## 專案結構

- `/C#`: 包含 C# 版本的術科程式原始碼。
  - `第一套`: 實作 for, while, do-while 三種迴圈版本的獨立專案。
  - `第二套`: 術科第二套試題 (1060306, 1060307, 1060308)。
- `/VB`: 包含 VB.NET 版本的術科程式原始碼。
  - `第一套`: 實作 For, While, Do-Loop 三種迴圈版本的獨立專案。
  - `第二套`: 術科第二套試題 (1060306, 1060307, 1060308)，採用 MVP 架構。
- `/Test`: 自動化測試工具集
  - `TestLauncher`: 視覺化測試啟動器 (WinForm GUI)。
  - `Task1Tester`: 第一套專屬測試專案 (含迴圈型態檢查、GoTo 禁用檢查、及 C#/VB 註解格式校驗)。
  - `WinFormUITester`: 第二套專屬 UI 自動化測試專案 (基於 FlaUI)。

## 健全度驗證 (Stress Testing)
本系統之測試工具已通過人為製造錯誤的壓力測試，確保能精確識別以下違規情況：
- **UI 佈局驗證**：自動偵測視窗標題錯誤、應檢人資料（姓名、座號）不符、日期格式錯誤。
- **DataGridView 結構檢查**：驗證欄位名稱是否符合規範、欄位數量是否正確。
- **資料邏輯校驗**：
  - **1060306**: 強制驗證 ID_NO 字典排序及 A/B/C 三階段錯誤優先級。
  - **1060307**: 驗證發牌不重複邏輯，及純點數比大小勝負判定。
  - **1060308**: 驗證分數加減乘除結果之約分精確度，及整數不顯示分母之規定。

## 視覺化測試工具 (TestLauncher)

`TestLauncher` 是一個整合式的 WinForm 啟動器，讓應試人能以視覺化方式執行所有測試。

### 核心功能：
- **雙語言支援**：可自由選擇測試 C# 或 VB.NET 目標專案。
- **一鍵全自動執行**：在第二套分頁中提供「執行全部」按鈕，依序自動完成 06~08 的所有 UI 與資料校驗。
- **工具列語系切換**：透過選單即時切換繁體中文與英文介面。

### 執行方式 (開發模式)：
在專案根目錄執行：
```powershell
dotnet run --project Test/TestLauncher/TestLauncher.csproj
```

### 發佈與分發 (部署模式)：
本專案提供 `publish.bat` 批次檔，可一鍵將測試工具打包為單一可執行檔：
1. 執行根目錄下的 `publish.bat`。
2. 編譯完成後，將產生的 `dist/` 目錄壓縮分發給應試人。

## 測試工具細節 (核心邏輯)

### 1. 程式碼校驗器 (CodeValidatorService)
- **智慧型迴圈偵測**：自動排除註解中的文字，精確識別程式碼中使用的迴圈型態。
- **跨語言標頭檢查**：支援 C# (`//`) 與 VB (`'`) 的 30 顆星號格式標頭驗證 (Rule 5.1)。
- **禁用函式檢查**：嚴格禁止使用 `Math.Sqrt` 或 `Array.Sort` 等系統內建函式進行核心運算。

### 2. UI 自動化測試 (WinFormUITester)
- **動態回合數適應**：自動讀取測試檔第一行，動態驗證 DataGridView 列數。
- **全資料列校驗**：逐行讀取 DataGridView 數值，與內部模擬的技術士標準邏輯進行 100% 精度比對。

## 技術規格
- **開發平台**：.NET 10 (Windows Forms)。
- **自動化技術**：FlaUI.UIA3 / xUnit。
- **品質保證**：維持 **0 錯誤 0 警告** 的編譯狀態。

## 注意事項
- UI 測試執行期間，系統會模擬滑鼠與鍵盤操作，請勿移動滑鼠以免干擾。
- 請確保電腦中已安裝 .NET 10 執行環境。
