# 11900 電腦軟體設計丙級 - 自動化測試系統

本專案提供針對「電腦軟體設計丙級」試題的自動化檢測工具，旨在協助應試人確認程式碼邏輯與輸出結果是否符合評分標準。

## 專案結構

- `/C#`: 包含考生開發的術科程式原始碼。
  - `第一站`: 術科第一題至第五題。
  - `第二套`: 術科第二套試題 (1060306, 1060307, 1060308)。
- `/Test`: 自動化測試工具集
  - `TestLauncher`: 視覺化測試啟動器 (WinForm GUI)。
  - `Task1Tester`: 術科第一站專屬自動化測試專案 (含程式碼規範與 PDF 頁首校驗)。
  - `WinFormUITester`: 術科第二套專屬 UI 自動化測試專案 (基於 FlaUI)。
- `ans.pdf`: 術科測試參考答案 PDF。
- `119003B14.pdf`: 術科測試應試人參考資料。

## 視覺化測試工具 (TestLauncher)

`TestLauncher` 是一個整合式的 WinForm 啟動器，讓應試人能以視覺化方式執行所有測試。

### 核心功能：
- **一鍵全自動執行**：在第二套分頁中提供「執行全部」按鈕，依序自動完成 06~08 的所有 UI 與資料校驗。
- **全域應檢人資料**：頂部獨立區塊，輸入一次姓名、編號與座號，即可套用於所有測試。
- **工具列語系切換**：透過選單即時切換繁體中文與英文介面。
- **彩色 Log 輸出**：仿 PowerShell 風格，直觀區分執行過程。
- **高度相容性**：針對不同視窗比例進行佈局優化，解決元件重疊問題。

### 執行方式：
在專案根目錄執行：
```powershell
dotnet run --project Test/TestLauncher/TestLauncher.csproj
```

## 測試工具說明 (命令列介面)

### 1. 術科第一站自動化測試 (Task1Tester)
用於驗證 940301~940305 的演算法邏輯與 PDF 輸出格式。

### 2. 術科第二套 UI 自動化測試 (WinFormUITester)
使用 FlaUI 模擬真實使用者操作，驗證 1060306~1060308 的 UI 互動與資料呈現。

#### 核心特點：
- **全資料列校驗 (Data Integrity)**：不只檢查 UI 佈局，還會逐行讀取 DataGridView 中的數值，並根據專案邏輯進行二次運算比對 (如身分證檢查碼、分數簡約、撲克牌勝負)。
- **智慧型相對定位**：視覺座標演算法自動辨識標籤右側元件，支援 TextBox 與 Label 混合佈局。
- **異常資料處理**：自動正規化處理 `(null)` 或空字串，避免 UIA 回報值的誤判。
- **日期格式適應**：自動驗證第二套要求的 **西元年 (YYYY/MM/DD)** 格式。

#### 執行方式：
```powershell
# 執行所有 UI 測試
dotnet test Test/WinFormUITester/WinFormUITester.csproj
```

## 技術規格
- **開發平台**：.NET 10 (Windows Forms)。
- **自動化技術**：FlaUI.UIA3。
- **品質保證**：專案採 `#nullable enable` 規範，目前維持 **0 錯誤 0 警告** 的編譯狀態。

## 注意事項
- UI 測試執行期間，請勿操作滑鼠或鍵盤，以免干擾自動化流程。
- 本工具預設資料已對齊範例程式 (陳宇威 / 112590005 / 005)。
